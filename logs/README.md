# Communication Logs

This directory contains auto-committed cross-layer communication logs.

## Structure

```
logs/
├── YYYY-MM/
│   ├── YYYY-MM-DD.md           # Daily cross-layer messages
│   └── YYYY-MM-DD-decisions.md # Daily decisions
└── README.md
```

## What Gets Logged

- All cross-layer communications (L1↔L3, L2↔L1, etc.)
- Strategic decisions with context and rationale
- Escalation events and their resolutions

## Commit Frequency

- Every 20 cross-layer messages, OR
- Every 5 minutes (whichever comes first)
